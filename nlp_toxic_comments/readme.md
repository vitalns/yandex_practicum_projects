# Поиск токсичных комментариев

***Описание проекта:***<br/>
Интернет-магазин запускает новый сервис. Теперь пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Магазину нужен инструмент, который будет искать токсичные комментарии и отправлять их на модерацию. 

***Задача:***<br/>
Классифицировать комментарии на позитивные и негативные.


***Используемые библиотеки и инструменты:***<br/>
pandas, numpy, nltk, sklearn, catboost, pipeline, metrics, time, corpus (stopwords), linear_model (LogisticRegression), feature_extraction (TfidfVectorizer), model_selection (train_test_split, cross_val_score, StratifiedKFold).
